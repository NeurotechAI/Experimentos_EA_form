Participa en nuestro experimento sobre carga mental

El equipo NeuroTechAI estamos llevando a cabo un experimento para estudiar la carga mental durante tareas cognitivas. No necesitas ninguna preparación previa para participar.

Durante el estudio, se utilizarán los siguientes dispositivos para registrar tu actividad:

  • EEG: casco de registro de actividad cerebral
  • Eye-tracker: gafas para seguimiento ocular
  • ECG: electrodos para medir la actividad cardíaca
  • fNIRS: sensores para medir la oxigenación cerebral

El experimento consta de dos sesiones de 1 hora cada una. Se realizarán preferiblemente de forma consecutiva. Si no es posible, podrán hacerse por separado.

Antes de participar, se entregará un consentimiento informado que deberás firmar.

Información importante

  • No es necesario estudiar ni prepararse para la prueba.
  • No se permite el uso de teléfono móvil durante el registro.
  • El experimento se realizará en la ETSIT (Edificio B, aula B304).
  • Si usas gafas graduadas, contacta previamente con nosotros y acude con lentillas.
